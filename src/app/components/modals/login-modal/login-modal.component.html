Gestion de Usuario
Se requiere usuario.
El usuario debe tener un minimo de 5 letras.
El usuario debe tener un maximo de 12 letras.
El usuario solo puede contener letras y numeros.
Se requiere contraseña
La contraseña debe tener un minimo de 5 letras.
La contraseña debe tener un maximo de 12 letras.
La contraseña debe contener al menos 1 letra mayuscula, 1 letra minuscula y 1 numero.
